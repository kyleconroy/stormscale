Magic the Gathering: Mechanics Storm Scale Chart

Those of you who read Mark Rosewater's blog probably know the Storm Scale. If you don't, you should start reading it. It's pretty good. Anyway, the Storm Scale rates Maro's subjective estimation of how likely it is for a certain mechanic/theme/card to be (re-)printed in an expert expansion on a scale from 1 to 10:

  1. It’s definitely coming back one day.
  2. I would be shocked if we never see it again.
  3. I would be surprised if we never see it again.
  4. It will probably return one day.
  5. Slightly more likely to return than not.
  6. It could return if we find the right set for it.
  7. It could return but it’s far from sure.
  8. Unlikely but not impossible.
  9. Perhaps it could return if we find the perfect set.
  10. I highly doubt we will ever see it again.
